Netwroking & the Internet

Network Classification
LAN A collection of computers in a single building or building complex
MAN a medium size network
WAN world network links machines over great distances
Network Models
bus network machines connected to common communication line called the bus
star network one machine serves as central focal point to which all other machines are connected
Protocols
CSMA/CD Ethernet Carrier Sense, Multiple Access with Collision Detecton. THe protocol that gives the right to transmit a message on a bus network- dictates that each message is broadcast to all machines on the bus
CSMA/CA WiFi Carrier Sense, Multiple Access with Collision Avoidance - the protocol designed to avoid collisions- when collisions occur, the message must be retransmitted
Combining networks
repeater A device that passes signals between two busses using amplification without considering the meaning of the signals
bridge More complex and efficeint repeater that checks destination address and forwards a message only when destined for the computer on other side
switch A bridge with multiple connections
router Special purpose computers used for forwarding messages that provides links between networks
gateway The point at which one network is linked to an internet
Client/server model
P2P: peer-to-peer Interprocess comminication model that provides service to and receive service from each other -executed on temorary basis
distributed system Software systme that controls a network infrastructure that consists of software systems that execute processes on different computers
cloud comouting Huge pools of shared computers on the network allocated for use by clients as needed
Internet Addressing
Tier 1 High speed, high capacity, international WANs
tier 2 regional in scope and less pwoerful than tier 1
Tier 3 Intermediary ISP that provides access to the tier 1 and tier 2 cores
IP address
ICANN Internet Corporation for Assigned Names and Numbers nonprofit established to coordinate the Internet's operation - awards blocks of consectively numbered IP addresses
DNS Domain Name System - name servers used as Internet-wide directory system
TLD: Top-level domain Top-Level Domains - used to reflect the domains classification
Country-code TLDs Two-Letter TLDs specific for each country
Internet protocols
FTP File Transfer Protocol - application for accessing another computer from a great distance
HTTP Hyper Text Transfer Protocol - a more powerful protocol to handle today's web servers and browsers
SMTP Simple Mail Transfer Protocol - defines a way that two computers on the networkmay interact when tranmitting an email message from one hos tto the other
World Wide Web
URL Unifrom Resource Locator - unique address given to locate and retrieve documents on the web - has information need by a browser to contact the proper server
client-ide activities activity required by a browser or web server for a client to interact with the page or user
server-side activites activity required by a browser or web server for a server to interact with the page or user
Internet four-layer-software
application layer software units that use clients and servers that use Internet communication to carry out their tasks
transport layer used to send and recieve messages over the Internet - divides long messages into smaller segments
network layer decides in which direction a packet should be sent at each step through the path along the Internet
link layer transfers the packet - deals with the communication details of particular to the network
Internet is
packet-switching network the internet is a packet-switch system which digital data is sent by breaking data into packets which contain data and control information
fault tolerant ability of a system (network) to continue operating without interuption when components fail
redundant there is a backup for delivery when congestion or failures occur - increases reliability and helps scale to more people and devices
Cybersecurity
maleware malacious software that can be transferred to and executed on a computer
virus software that infects a computer by inserting itself into programs that already reside on the machine - when the host program runs, the virus executes.
Trojan Horse enters a computer system disguised as a desirable program - they are often in form of email attachment
spyware (sniffing software) - software that collects information and activities on a computer and reports back to the instigator
phishing obtaining information by asking for it - email and telephone cons are examples - usually perpetrator claims to be financial institution
DDoS attack Distributed Denial of Service attack - overloading computer with messages - launched against webservers to disrupt business
Protection
Firewall software that filters traffic passing through a point on a network
spam filter variation of firewall designed to block unwanted email
Proxy server software that acts as intermeidiary between a client and server with the goal of shielding the client from adverse actions of the server
antivirus software software that is used to detect and remove the presence of known viruses and other infections
cryptography study of sending and receiving messages securely in the presence of adversaries
public key encryption techniques where the encryption and decryption algorithms are known, and one key ios known, but the encoded message remains secure
