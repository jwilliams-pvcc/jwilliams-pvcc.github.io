The Major PLanets in Our Solar System

operating system The software that supports a computer's basic functions
job queue a list of tasks to be carred out in turn by an operating system
FIFO First-in first-out, is a validation method in which the assets produced or acquired first are sold, used, or disposed of first
interactive processing The process of interacting directy with a program and/or data for immediately modifying, retriecing and/or displaying
time-sharing the operation of a computer system by several users for different operations at the same time
multiprogramming a rudimentary form of parallel processing in which several programs run at the same time on a uniprocessor system
application software a computer program designed to carry out a specific task other than one relating to the operation of the computer itself
system software A typer of computer program that is designed to run a computer's hardware and application programs
utility software software designed to help analyze, configure, optimaize or maintain a computer
graphical user interface (GUI) a form of user interface that allows users to interact with electronic devices through graphical icons and audio
shell a computer program that exposses an operating system's services to a human user or other programs
kernel a computer program at the core of a computer's operating system and generally has complete control over everything in the system
device driver a special kind of software program that controls a specific hardware device attached to a computer
file manager program that provcides a user interface to manage files
memory manager a software utility that operates in conjunction with the operating system
virtual manager a tool or resource that allows for the managment of more than one computer from a single workstation
boot strapping any test or metric that uses random sampling with replacement, and falls uner the broaders class of resampling methods
ROM a non-volitalie memory type. (Read-Only-Memory)
deadlock a situation in which two computer programs sharing the same resource are effectively preventing each other from accessing the resourse, resulting in both programs ceasing to function
spooling Send (data that is intended for printing or processing on a peripheral device) to an intermediate store
superuser (or administrator) a superuser is an account with broad privileges, making it useful for system administration
multi-factor authenticating an authentication method that requires the user to provide two or more verification factors to gain access
